## 💡 Customization Tips

### Change Text/Content
1. Open any .html file in text editor
2. Find the text you want to change
3. Update it
4. Save file
5. Refresh browser to see changes

### Change Colors
- Edit `css/style.css`
- Search for color names: `--teal`, `--navy`, `--amber`, `--green`
- Change the hex values

### Add New Page
1. Create new .html file in pages folder
2. Copy content from similar page
3. Update links to match new page name
4. Add link to navbar/footer

### Add Real Content
- Replace placeholder text with real content

## 🚨 Common Issues & Fixes

### Issue: Pages won't load
**Check:** Are you using correct file paths?
- From root: `pages/courses.html` ✓
- From /pages/: `../index.html` ✓

### Issue: Images missing
**Check:** Is logo.jpg in assets folder?
- Logo should be at: `assets/logo.jpg`

### Issue: Mobile menu stuck
**Clear browser cache:**
- Ctrl+Shift+Delete (Windows)
- Cmd+Shift+Delete (Mac)
- Or try different browser

### Issue: Styling looks wrong
**Make sure CSS is loading:**
- Check browser console (F12)
- Verify css/style.css exists
- Clear cache and refresh
